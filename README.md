# UNIP - Jogo da forca

## Requisitos para executar

* NodeJs - versão v12.x
* NPM - versão v6.14.10

---
## Como executar 

No terminal execute o comando ``npm run start``

---
## Docker 

```
// Dentro do terminal

docker build -t <nome da imagem>

docker images

docker exec -it <containerID> bash

npm run start
```
